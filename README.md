# 🚀 Floating-Task-List (Tujiu is watching you!)

一款桌面时间管理挂件。

## ✨ 核心功能
- **秃鹫监工**：界面上方实时显示 "Tujiu is watching you"，时刻提醒你战胜拖延。
- **极致透明**：支持 0-100% 透明度调节，完美融入任何桌面壁纸。
- **自由交互**：无边框设计，按住顶部或底部可随意拖动位置。
- **个性化定制**：内置多种高冷字体颜色（深空黑、皓月白、深海蓝等）一键切换。
- **纯净体验**：精致的 Mac 风格半透明滚动条，不遮挡视线。

## 📥 如何下载使用 (绿色免安装版)

这款软件无需安装，解压即用，非常方便：

1. 点击页面右侧的 **[Releases](https://github.com/Tujiu118/Floating-Task-List/releases)**。
2. 下载最新的压缩包 `Tujiu-is-watching-you-v1.0.0-Portable.zip`。
3. **解压** 整个文件夹到你电脑的任意位置。
4. 进入文件夹，找到 **`我的桌面挂件.exe`** (或你重命名后的名字)，双击即可启动。
5. **小贴士**：你可以右键点击这个 `.exe` 文件，选择“发送到 -> 桌面快捷方式”，以后在桌面上就能一键呼唤秃鹫监工了！

## ✨ 核心功能
- **秃鹫监工**：界面上方实时显示 "Tujiu is watching you"，时刻提醒你战胜拖延。
- **极致透明**：支持 0-100% 透明度调节，完美融入任何桌面壁纸。
- **自由交互**：无边框设计，按住顶部标题或底部信息栏可随意拖动位置。
- **内容滚动**：中间任务列表支持鼠标滚轮滚动，并配有精致的 Mac 风格半透明滚动条。
- **颜色定制**：内置 5 种高冷预设字体颜色，在设置面板中一键切换。

## 🛠️ 技术栈
- **前端**: React 19 + Tailwind CSS
- **桌面框架**: Electron
- **动画**: Framer Motion
- **AI 辅助**: Google Gemini (Build Mode)
